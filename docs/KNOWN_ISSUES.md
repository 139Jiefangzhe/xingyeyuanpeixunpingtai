# 已知问题与技术债务

> 本文档记录当前版本所有已知缺陷、安全缺口和待优化项。
> 每次迭代前应优先查看 P0 项，完成修复后从本文档移除。

## P0 — 生产部署前必须修复

- [ ] **JWT 鉴权缺失**：当前 `X-User-Id` header bypass，任何人可伪造用户身份
- [ ] **CORS 开放**：`LocalCorsConfig` 使用 `*` 或本地宽松白名单，生产环境必须收紧为精确域名
- [ ] **文件上传无校验**：无类型/大小/病毒扫描
- [ ] **Feign Fallback NPE 风险**：部分调用方未统一判空，需逐条收口

## P1 — 上线前 2 周修复

- [ ] 操作审计日志缺失
- [ ] 敏感数据脱敏未确认
- [ ] 考试并发提交无分布式锁保护（autoGrade 阶段）
- [ ] 接口限流未配置（Sentinel 无规则）
- [ ] 防作弊未实现（切屏/IP/乱序均为预留接口）
- [ ] 前端路由鉴权绕过风险（`private-route` bypass）

## P2 — 后续迭代

- [ ] H2 ↔ MySQL 语法漂移
- [ ] Flyway 版本号冲突（各服务独立编号）
- [ ] 配置漂移（local/dev/prod 三套配置未统一）
- [ ] 日志未聚合（无 ELK/Loki）
- [ ] 监控告警未配置（Prometheus + Grafana 无落地）
